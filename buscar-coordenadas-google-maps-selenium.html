Buscando coordenadas geográficas no Google Maps com Selenium

Tue 10 Mar 2020

Nesse projeto fiz um script em Python usando a biblioteca do Selenium para buscar coordenadas geográficas no Google Maps.

A forma de capturar as coordenadas foi direto na URL, assim que buscado o endereço no campo de busca. Procurando mais tarde, acho que seria mais interessante usar uma API para essa mesma tarefa, podendo ser a do próprio Google Maps. De qualquer forma a ideia foi usar o Selenium :)

Os endereços foram tirados de uma planilha Excel e depois sendo gerada uma planilha com as coordenadas coletadas. Os arquivos estão no meu repositório.
